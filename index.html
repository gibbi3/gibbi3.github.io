PowerBall Checker

Powerball Checker is a rather cumbersome program that allows the user to input their Powerball numbers as arrays, and compare them at drawing time to the numbers that are drawn.
A method allowing the input of numbers bought seems reasonable; however, it was only needed for one occasion and was found to be sufficient, serving its anticlimactic purpose wondrously.
